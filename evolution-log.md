### 核心功能

1. **基础卡片：** 多段文本、标题、副标题
2. **交互卡片：** 按钮、链接、跳转
3. **多媒体卡片：** 图片、附件、音频
4. **高级样式：** 颜色、对齐、字体

### 使用方法

```python
from card import FeishuCardGenerator

card_gen = FeishuCardGenerator()

# 创建基础卡片
card = card_gen.create_message_card(
    title="通知",
    content=["这是一条消息", "这是第二条消息"]
)

# 创建按钮卡片
card = card_gen.create_button_card(
    title="操作确认",
    content="请选择操作",
    buttons=[
        {"text": "确认", "url": "https://..."},
        {"text": "取消", "action": "cancel"}
    ]
)
```
