# 安全扫描技能

提供漏洞检测、安全加固和入侵检测的安全扫描工具。

## 功能特性

- ✓ 端口扫描和开放端口检测
- ✓ 服务版本识别
- ✓ 常见漏洞检测
- ✓ 弱密码检测
- ✓ 安全配置检查
- ✓ 入侵检测日志分析
- ✓ 安全报告生成
- ✓ 自动化安全加固建议

## 安装

无需额外依赖，使用Python标准库。

## 快速开始

### 端口扫描

```bash
# 扫描常见端口
python port_scanner.py --target 192.168.1.1

# 扫描指定端口范围
python port_scanner.py --target 192.168.1.1 --ports 1-65535

# 高速扫描（更多线程）
python port_scanner.py --target 192.168.1.1 --threads 500

# 保存结果
python port_scanner.py --target 192.168.1.1 --output scan_results.json
```

### 漏洞扫描

```bash
# 基础扫描
python vuln_scanner.py --target 192.168.1.1

# 使用端口扫描结果
python vuln_scanner.py --target 192.168.1.1 --open-ports scan_results.json

# 生成HTML报告
python vuln_scanner.py --target 192.168.1.1 --output report.html --format html
```

## 扫描类型

### 端口扫描

- **快速扫描：** 常见端口（1-1024）
- **完整扫描：** 全端口（1-65535）
- **自定义扫描：** 指定端口列表

### 漏洞扫描

- **基础扫描：** 常见漏洞检测
- **完整扫描：** 深度漏洞分析

## 常见漏洞检测

| 漏洞类型 | 端口 | 严重性 | 描述 |
|---------|------|--------|------|
| SSH弱认证 | 22 | Medium | SSH可能存在弱认证风险 |
| HTTP旧版本 | 80 | Low | HTTP服务版本过旧 |
| FTP匿名登录 | 21 | High | FTP允许匿名登录 |
| Telnet不安全 | 23 | High | Telnet是不安全协议 |
| MySQL默认凭证 | 3306 | High | MySQL可能使用默认凭证 |
| RDP漏洞 | 3389 | High | RDP服务可能存在漏洞 |
