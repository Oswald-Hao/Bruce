# 电商套利系统

## 技能描述

**赚钱能力：⭐⭐⭐⭐⭐**

自动监控多个电商平台（淘宝、拼多多、京东、亚马逊）的商品价格，发现价格差异（套利机会），帮助用户通过低价买入、高价卖出赚取差价。

## 核心功能

### 1. 多平台价格监控
- 支持淘宝、拼多多、京东、亚马逊等主流平台
- 自动搜索指定关键词的商品
- 多线程并发搜索，提高效率

### 2. 智能套利分析
- 自动识别相同/相似商品
- 计算价格差异和利润率
- 筛选高利润套利机会（默认>10%）

### 3. 实时机会通知
- 发现套利机会立即通知
- 显示买入/卖出平台、价格、链接
- 计算预期利润

### 4. 数据持久化
- 自动保存套利机会到JSON文件
- 支持历史记录查询

## 赚钱方式

### 方式1：代购赚差价
- 在低价平台（拼多多）购买商品
- 在高价平台（闲鱼）卖出
- 赚取价格差价

### 方式2：倒卖套利
- 批量低价买入热门商品
- 加价10-20%转卖
- 适合有资金的用户

### 方式3：信息变现
- 将套利机会信息出售给专业卖家
- 按条收费或月度订阅

## 使用方法

### 基础使用

```bash
cd /home/lejurobot/clawd/skills/ecommerce-arbitrage
python3 arbitrage.py
```

### 自定义关键词

修改arbitrage.py中的keywords列表：

```python
keywords = [
    "iPhone 15",
    "AirPods Pro",
    "你的商品",
]
```

### 调整利润率阈值

修改ArbitrageAnalyzer的min_profit_rate：

```python
analyzer = ArbitrageAnalyzer(min_profit_rate=15.0)  # 只显示>15%的机会
```
