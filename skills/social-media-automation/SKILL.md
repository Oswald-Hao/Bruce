# 社交媒体自动化系统 - SKILL.md

## 技能描述

社交媒体自动化管理系统，支持多平台内容发布、定时发布、互动管理和数据分析。

## 核心功能

### 1. 多平台内容发布
- 支持微信公众号、抖音、小红书、微博、知乎等平台
- 批量发布内容到多个平台
- 自动适配不同平台的格式要求
- 支持图片、视频、文章等多种内容类型

### 2. 定时发布系统
- 设置定时发布任务
- 批量安排发布计划
- 自动执行发布任务
- 发布状态追踪

### 3. 互动管理
- 自动回复评论和私信
- 点赞和关注策略
- 粉丝管理工具
- 互动数据统计

### 4. 内容管理
- 内容库管理
- 模板系统
- 快速编辑工具
- 多媒体资源管理

### 5. 数据分析
- 粉丝增长分析
- 内容效果分析
- 互动数据分析
- 对手数据分析

## 工具脚本

### main.py
主程序入口，提供命令行接口。

### platforms/
各平台适配器：
- wechat_mp.py - 微信公众号
- douyin.py - 抖音
- xiaohongshu.py - 小红书
- weibo.py - 微博
- zhihu.py - 知乎

### scheduler.py
定时发布调度系统。

### analytics.py
数据分析模块。

## 安装依赖

```bash
pip install requests
pip install schedule
pip install pandas
```

## 使用示例

```bash
# 发布内容到多个平台
python main.py publish --content "Hello World" --platforms douyin,xiaohongshu

# 设置定时发布
python main.py schedule --content "New Article" --time "2026-02-09 10:00" --platforms douyin

# 查看发布统计
python main.py stats --days 7

# 自动回复管理
python main.py autoreply --enable --template "template.json"
```

## 配置文件

config.json - 存储各平台的API密钥和配置信息。
